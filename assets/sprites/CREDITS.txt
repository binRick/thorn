Thorn — sprite set (CC0 1.0)
============================

These sprite strips are ORIGINAL art for Thorn, generated by the game itself
(`./build/thorn --gen-assets`; see GenAssets() in src/main.c) and released into
the public domain under CC0 1.0. Use, modify, and redistribute freely.

Format (the drop-in convention)
  - Horizontal PNG strips of equal-width frames.
  - hero_idle.png  : 1 frame      hero_walk.png : 2 frames
  - skarl.png / brute.png / sentry.png / maldrak.png : 1 frame each (enemies)
  The loader (LoadStrip in src/main.c, called from InitSprites) reads these on
  startup if present; if a file is missing it falls back to the in-code
  generated sprite, so the game always runs.

Reskinning with another set
  Replace these PNGs with your own art, or any permissively-licensed pack (e.g.
  CC0 sets from kenney.nl or opengameart.org). Keep the same filenames; if your
  strips have a different frame count, update the count in the LoadStrip(...)
  calls in InitSprites(). If you use third-party art, record its source and
  licence here (CC0 needs no attribution, CC-BY does).
